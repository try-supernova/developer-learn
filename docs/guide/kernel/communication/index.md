Pluto processes may communicate with the kernel in two ways, described below.

| Name                                                           | Type                       | Compare to... |
|----------------------------------------------------------------|----------------------------|---------------|
| [Call](/guide/kernel/communication/calls)                      | Request-response           | HTTP          |
